A sample Python project
=======================

This a utils and toolbelt project to wrap some of the openCV functionality in python.

----

To install and use:

Clone the project.
```
git clone https://github.com/dataloudlabs/cv_utils.git
```

Install it using pip in editable mode.
```python
pip install -e
```
